دليل المستخدم - نظام إدارة الموارد البشرية

1. تسجيل الدخول:
   - افتح المتصفح وانتقل إلى: yourdomain.com/index.php
   - أدخل اسم المستخدم وكلمة المرور الخاصة بك كمشرف.

2. لوحة التحكم:
   - ملخص الحضور، عدد الموظفين، المهام النشطة، والتقارير.

3. إدارة الموظفين:
   - إضافة، تعديل، حذف موظف.
   - تحميل المستندات الخاصة بالموظفين.
   - تتبع الحضور والإجازات لكل موظف.

4. الحضور:
   - استيراد حضور من جهاز البصمة.
   - إدخال يدوي للحالات الخاصة.
   - عرض وتحليل التأخير والغياب.

5. المرتبات:
   - يتم احتساب المرتب تلقائيًا بناءً على الحضور والخصومات والمكافآت.
   - تصدير المرتبات إلى Excel وPDF.

6. المهام:
   - تعيين مهام للموظفين ومتابعة تنفيذها.

7. التنبيهات:
   - عرض تنبيهات تلقائية عن التأخير، المهام غير المنجزة، الغياب المتكرر، إلخ.

8. الصلاحيات:
   - تخصيص صلاحيات المشرفين حسب النظام أو الصفحة.

9. التقارير:
   - تقارير يومية وشهرية لكل موظف أو فرع.
