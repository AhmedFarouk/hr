## Configuration

- Database: `config/db.php`
- Constants: `config/constants.php`

## Authentication

- Session is managed in `includes/auth.php`
- All pages include permission checks using `has_permission()`

## Database Installation

- Run `installer/installer.php` or import `installer/schema.sql`

## APIs

- `api/employees.php` - returns employee data in JSON
- `api/attendance.php` - returns attendance logs

## Export

- `salaries/export_excel.php` uses PHPSpreadsheet
- `salaries/export_pdf.php` uses TCPDF

## Notifications System

- Configurable from `notifications/settings.php`
- Triggered on:
  - Late attendance
  - Missed tasks
  - Excessive absences

## Customization

- Themes: `assets/css`
- Components: `includes/header.php`, `includes/footer.php`

## Deployment

- Upload project to `/public_html/` or desired directory
- Ensure `uploads/` folder is writable (chmod 755 or 777)
- Set correct DB credentials in `config/db.php`

---

## Security Notes

- CSRF protection is recommended for forms
- Use HTTPS in production
